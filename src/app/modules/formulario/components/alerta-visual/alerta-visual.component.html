Alerta Visual

Podemos apresentar alertas visuais para os campos que exigem algum tipo de validação, como "required". No exemplo a seguir, vamos adicionar um alerta para o usuário quando o campo "input" for tocado ou alterado. Também vamos adicionar a diretiva "minlength" para adicinar uma validação de quantidade mínima de caractéres.

*O Campo deve ser preenchido. *O texto deve ter no mínimo 5 caracteres.
