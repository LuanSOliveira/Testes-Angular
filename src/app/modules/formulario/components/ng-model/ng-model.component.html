ngModel

A diretiva "ngModel" é usada para vincular os campos que serão validados ao formulário, ele é referenciado junto a propriedade "name" da tag que foi atribuído. Neste exemplo, a tag "input" está recebendo o "name" de "textoNome" e a diretiva "ngModel" para que quando o formulário for submetido, seja impresso no console as informações dos campos que estão sendo controlados pelo "formDir". Ou seja, os campos com o "ngModel".
