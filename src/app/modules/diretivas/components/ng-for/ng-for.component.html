*ngFor (Renderização de listas)

Com a diretiva "ngFor" podemos iterar sobre um array e renderizar elementos de acordo com a quantidade de itens desse array. Também podemos nos utilizar dos valores dos elementos desse array com a INTERPOLAÇÃO DE DADOS. No exemplo a seguir estamos renderizando um card para cada elemento do array "pessoas" e renderizando as informações dinamicamente com os valores dos elementos.

  • Nome: {{pessoa.nome}}

    Número: {{pessoa.numero}}
