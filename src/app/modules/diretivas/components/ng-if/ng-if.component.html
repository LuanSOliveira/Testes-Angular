*ngIf (Renderização Condicional)

A diretiva "ngIf" é responsável por gerar a renderização condicional das tags html. No exemplo a seguir o texto "Condição Verdadeira" só será apresentado quando o valor da variável "condicao" for "true". O evento de "click" no botão executa uma função que altera o valor da variável "condicao".
